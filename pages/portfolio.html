Personal Finance Dashboard

The Personal Finance Dashboard is a project crafted using Microsoft Power BI and Google Sheets. This dashboard empowers users to visually track their income, expenses, and savings data, providing insights into their financial situation.

All transactions are conveniently stored in a Google Sheet, seamlessly connected to Power BI for auto-refreshing purposes. This dynamic setup ensures hassle-free tracking of expenses, allowing users to pinpoint where their money goes. Moreover, it aids in monitoring saving goals and offers insights into income and expense trends over specific periods. It also keeps a watchful eye on your Net Worth.

Sales Insigts Dashboard

AtliQ Hardware is a company that supplies computer hardware and peripherals to numerous clients. With a Head Office located in Delhi and several Regional offices, the company faces challenges in gaining insights for each region due to the dynamic growth of the market.

The Sales Insights Dashboard, created using Microsoft Power BI and MySQL, aims to address these challenges by providing the management of AtliQ Hardware with tools to analyze the performance of their business.

The data for this dashboard is sourced from codebasics' GitHub page, enabling analysis of revenue and sales data for AtliQ Hardware over a specified period.

Cricket Analytics Dashboard

This repository contains a Cricket Best XI Dashboard created using Microsoft Power BI and Python. This Dashboard aims to select the best players based on a set criteria. Data can be scraped or can be retrieved from the JSON files provided.

The Dashboard is divided into different player positions such as Openers, Middle Order, Finishers, All Rounder and Fast Bowlers. Each player position has certain criteria set so that we can select the optimal players in each position.
